## Executing Programs
### Compiling
---
To compile a Rust program using the `rustc` command, run:
```
rustc file.rs
```
this will create an executable file `file` in the current directory.
### Running
---
To run a compiled Rust program in the terminal, run:
```
path/to/compiled_file_name
```
